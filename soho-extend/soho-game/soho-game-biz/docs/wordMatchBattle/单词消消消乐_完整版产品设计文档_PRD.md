# 单词消消消乐（Word Match Battle）
## 完整版产品设计文档（PRD）

---

## 1. 文档信息
- 产品名称：单词消消消乐（Word Match Battle）
- 文档类型：完整版 PRD
- 产品类型：学习型消除 + 多人竞技
- 目标平台：Web / 小程序 / App
- 版本号：V1.0

---

## 2. 产品背景与机会

### 2.1 行业背景
- 英语词汇学习长期存在“枯燥、低效”的问题
- 消除类游戏用户基数大、学习成本低
- 家长和学生对“寓教于乐”接受度高

### 2.2 核心机会
> 将“单词记忆”嵌入高频、高刺激的对战消除循环中

---

## 3. 产品目标

### 3.1 商业目标
- DAU 稳定增长
- 赛季制提升留存
- 非破坏公平性的商业化

### 3.2 学习目标
- 高频词汇多场景曝光
- 错词强制复现
- 可量化学习效果

---

## 4. 用户画像

| 用户类型 | 特征 |
|------|------|
| 学生 | 有明确词汇量目标 |
| 家长 | 关注学习效果 |
| 泛游戏用户 | 喜欢竞技 |
| 自学者 | 碎片化学习 |

---

## 5. 核心玩法设计

### 5.1 棋盘设计
- 8×8 格子
- 单词块 / 字母块 / 干扰块

### 5.2 消除规则
- 拼词消除
- 翻译消除
- 同义词消除
- 词性匹配

### 5.3 错误惩罚
- 扣能量
- 投放自我干扰

---

## 6. 学习系统设计

### 6.1 词库体系
- 分级词库（中考 / 高考 / CET / IELTS）
- 动态难度调整

### 6.2 记忆曲线
- 首次：释义提示
- 二次：无提示
- 三次：反向测试

---

## 7. 多人竞赛系统（核心）

### 7.1 模式设计
- 快速对战（1v1）
- 排位赛
- 4 人混战
- 好友房

### 7.2 对战机制
- 双棋盘
- 连击干扰
- 高级词攻击
