# 手机认证模块

    该模块主要是对接手机相关认证功能。  包括 接收短信验证码认证， 拨打电话获取认证信息， 拨打电话语音验证码

## 模块架构设计

    服务器端提供基础的api入口。 通过API接口同短信网关通信进行收发短信，拨打，接听电话。 本系统附带一份基础的  android系统相关实现。

## 接收短信验证码认证

    我们常见的短信认证都是发送短信到客户手机。 这样的确很方便， 但是有个问题是， 短信费用是系统方提供的， 按照市场价大约4分一条，如果客户量不少的话成本也是相对较高的。
    在系统用户快速增长时期或者长期让用户主动发送短信到指定短信网关可以节约一定的成本。

## 接收电话告知认证信息

    用户主动拨打电话到指定号码， 接通告知认证信息，相对来讲可能麻烦点， 但是能够节约成本 并且能够更好的提高系统安全性，相对来讲是一种不错的选择

## 系统拨打用户电话告知认证信息

    这种方式相对来讲， 可能需要一定的运营费用， 相对来讲现在的话费比较便宜 甚至比短信还要经济很多， 也是一种不可多得的认证方式， 可以有效的人机互动，减少机器人。
